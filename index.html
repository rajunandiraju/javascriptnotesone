Rules for Variables?

  • No Uppercase Initial Letter
  • Meaningful and Descriptive Variable Name

DataTypes in JavaScript

  • number : Number or Decimal Values
  • string : Sequence of Characters
  • Boolean : True or False to take decisions
  • undefined : Value taken by a Variable that is not defined.
  • null : Empty value. (Used in Different circumstances)
  • symbol : Define Unique Value (can't be changed)
  • big int : Integers that are too large to be represented by the number type.

Dynamic Typing

The ability of JavaScript to automatically determine the data type of the value stored in a variable.

What are Comments?

Single Line Comment: Content is written in between //.
Multi Line Comment: Content is written in between /* & */

Special Operators (typeof)
+ operator or - operator that we can use to show the type of a value.

Types of Variables?

  • Let - Values that can change the data
  • const - used to declare variables that aren't supposed to change in the future. To reduce Errors.
  • Similar to Let variable. Used in Different circumstances.
